### Installations
```
nstall nodejs
npm install mysql-server
```
### Setup

Import the database
  - `create database matrimony` 
  - `mysql -u root -p matrimony < /path/to/this/repo/matrimony.sql`
### Start Server
```
cd /path/to/app/directory
node app.js
```
### Launch App
```
visit http://localhost:3000/
```
